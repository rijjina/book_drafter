# Source Map And Authority

## Selection Rule

Choose one primary governing source for the project. Record its title, issuing institution, year, applicable document type, and any local amendments. Treat the remaining sources as supporting guidance.

The public distribution does not bundle the original PDFs because redistribution rights may vary. The distilled rules in the other reference files are available by default. Add institution- or publisher-specific source files to the project and use `refresh-sources` when exact wording or a newer governing rule is required.

## Cautions

- `ประกาศประเมินการสอน.pdf` is image-based. When the user supplies it, verify exact wording visually in the original pages.
- Publisher manuals contain house-style rules. Do not treat their font, margin, or citation details as universal law.
- Practical page-count advice in presentation materials is indicative, not a substitute for current institutional regulations.
- When a newer institutional rule is supplied, use it and document what it supersedes.
